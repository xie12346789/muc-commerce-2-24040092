## 用自己的话回答

- 什么是特征，什么是标签：特征是用来预测的输入信息，如用户的使用月数和投诉记录；标签是要预测的目标答案，如用户是否流失。
- 为什么要保留测试集：测试集用来模拟模型没有见过的新数据，评估模型的泛化能力，避免过拟合。如果只用训练集评估，模型可能在训练集上表现很好但在新数据上表现很差。
- 为什么83%准确率仍可能没有用：因为数据存在类别不平衡，流失率只有约17%，最低参照线（永远预测不流失）的准确率也能达到83%。这说明模型可能什么都没学到，只是简单预测多数类。此时需要看召回率等更全面的指标。
