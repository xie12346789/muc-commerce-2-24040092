## 核心功能

### 1. 登录系统
- 演示账号：`student` / `day07`
- 正确账号可进入数据看板
- 错误密码显示明确提示
- 未登录访问 `/dashboard` 自动跳转登录页
- 退出后 Session 被清除

### 2. 数据看板
- 4张指标卡：总用户数、流失用户、总体流失率、平均订单数
- 2张图表：品类用户比较柱状图、用户生命周期流失趋势折线图
- 1张数据表：偏好品类用户统计
- 数据观察：自动生成生命周期风险分析

### 3. 品类筛选
- 支持按偏好品类筛选数据
- 选择"全部"时显示所有品类
- URL 参数、下拉框与表格结果保持一致

### 4. 离线问答
支持以下5类问题：
- 总体规模：系统中有多少用户？
- 流失情况：总体流失率是多少？
- 偏好品类：哪个品类用户最多？
- 生命周期：哪个阶段风险最高？
- 订单情况：平均订单数是多少？

## 必选拓展

### 拓展B：生命周期详情页

**访问地址**：`http://127.0.0.1:5000/segments`

**功能说明**：
- 展示生命周期阶段的详细指标：用户数、流失人数、流失率、平均订单数、平均优惠券数、平均返现、平均距上次下单天数
- 数据观察：自动计算最高/最低流失率阶段，并显示两者差距
